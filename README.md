# Contador Simple con React

Este es mi proyecto de contador de segundos realizado con React. El objetivo principal fue crear un componente que muestre el tiempo transcurrido desde que la página se carga, practicando el uso de props, intervalos y renderizado dinámico.

## Lo que incluye el proyecto:
- **Componente SecondsCounter:** Un componente reutilizable que recibe los segundos por props.
- **Iconografía:** Uso de Font Awesome para el icono del reloj.
- **Estilos:** Diseño oscuro con cajas individuales para cada dígito.

## Funcionalidades Bonus (Completadas):
- **Botones de Control:** Añadí lógica para poder pausar el contador, resumirlo y reiniciarlo a cero.
- **Alerta Personalizada:** Al cargar la página, el sistema te pregunta en qué segundo quieres recibir una alerta. Al llegar a ese tiempo, se dispara un aviso en el navegador.

## Cómo ejecutarlo:
1. Clonar el repo.
2. Instalar dependencias: `npm install`
3. Correr el proyecto: `npm run start`
